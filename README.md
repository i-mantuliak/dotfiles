# dotfiles
My Dotfiles

settings for XDG envs:
```shell
cp .zshenv ~/
```

nvim setup:
```shell
brew install neovim ripgrep fzf node@20 bat
echo 'export PATH="/opt/homebrew/opt/node@20/bin:$PATH"' >> ~/.zshrc
```
ghostty setup
```shell
brew install --cask ghostty
brew install --cask font-jetbrains-mono-nerd-font
```
